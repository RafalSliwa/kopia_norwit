Below are some notes you should pay attention to while using Contact Form Ultimate:

Contact Forms

When creating a new contact form, if you click on "Add new input field/row to create your contact form" link and select an input field, by default this input field will be put on a full-width row (1 column).

To add reCAPTCHA input field, you need to enable reCAPTCHA first. Navigate to "Settings > Integration > reCAPTCHA", enable reCAPTCHA option and enter your key pair.

For how to get reCAPTCHA site key and secret key, please read our module documentation

To be able to reply customer messages directly on back office, you need your customer email address. When building a contact form, make sure to add an email input field and mark it as required field.

Email configurations

To get the info from your contact form to the email send to admin or auto responder email, please copy and paste the respective mail-tags into "Message body" field in "Contact Forms > Mail" subtab

To receive attachment file from your customer via email, please navigate to "Contact Forms > Mail" and check the "File attachment" box.

To receive attachment file from your customer via “Messages” tab, please navigate to "Contact Forms > Settings" and turn on the "Save attachments" option.
