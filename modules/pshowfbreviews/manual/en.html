To generate an Access Token:

I. Adding an application (if you have an application on Facebook, skip this point):
1. Go to https://developers.facebook.com/docs/apps/.
2. Click on "My Apps" in the upper right corner of the page.
3. Click on "Create New App".
4. Follow the instructions on the screen.
II. Downloading the application ID:
1. Select the newly added application.
2. At the top, copy the number named "APP ID:" and paste it in the module settings.
III. Downloading the secret application key:
1. Select the newly added application.
2. Select Settings-> Basic. From the menu on the left.
3. In the "App Secret" field, click on the "Show" button and copy the code from this field into the module settings.
IV. Creating a token for a page:
1. Open graph Explorer: https://developers.facebook.com/tools/explorer/.
2. Choose your application from the right corner dropdown.
3. From "Get Token" dropdown choose your Fan Page.
4. Click on submit button to generate token.
5. From the left side on "Search for a field" enter access_token and click submit again. Copy this token from the main window.
6. Open https://developers.facebook.com/tools/debug/accesstoken and paste token here. Click "Debug".
7. Click the button "Extend Access Token". This will generate never expiring token.
V. Getting the page ID:
1. Go to https://developers.facebook.com/tools/explorer/?method=GET&path=me&version=v4.0.
2. Then select your page on fb and enter only the word "me" in the input.
3. At the end, click the submit button.
